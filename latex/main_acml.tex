\begin{abstract}
In this paper, we propose a robotic system that integrates symbolic planning, uncertainty modeling, and user corrective feedback in a knowledge-based framework to reduce human intervention and workload. Existing human-in-the-loop robot systems often require continuous user involvement, which increases interaction frequency and workload. To address this challenge, we model the system as a Partially Observable Markov Decision Process (POMDP) and track uncertainty during planning and execution. We introduce an entropy-based quantitative measure to evaluate uncertainty and generate queries that focus on the sources of ambiguity. The system requests user feedback if the estimated uncertainty exceeds a threshold and updates the knowledge base accordingly. The proposed system is integrated into a real robotic platform. Experimental results show that it maintains a comparable task success rate to systems that rely on frequent user intervention, while reducing the number of queries.
\end{abstract}
