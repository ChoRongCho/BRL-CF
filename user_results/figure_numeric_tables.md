# Figure Numeric Tables

이 문서는 `domain_condition_friedman` 및 `sagat_level_friedman` 폴더의 figure를 다시 그릴 때 필요한 숫자값을 정리한다.
막대 그래프는 `mean`을 높이로, `se`를 error bar로 사용한다. 별표는 `p_bonferroni`와 `stars`를 사용한다.
